# Next.js App Router — Train Stations (Germany) — Leaflet Map

This project is converted to Next.js App Router (`app/` directory) and keeps the original functionality:
- Fetch stations from the provided Gist
- Display markers on a Leaflet map
- Filter by city, list + map interaction

Run:
1. npm install
2. npm run dev
3. open http://localhost:3000
